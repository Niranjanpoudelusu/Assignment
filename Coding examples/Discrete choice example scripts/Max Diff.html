Some intresting summary and notes from the analysis

  • Includes the count analysis which gives best worst scales and logit modeling.
  • In the modeling analysis Message 15 (Flexible dosing) is used as base/reference study (Can be chosen anyone, just to make all the estimates positive).
  • For the purpose of random parameter estimates only 10 draws are taken for economic computation.
  1. Message 4 (Efficacy in high baseline VL) is the relatively most preferred option.
  2. Message 15 (Flexible dosing) is relatively least preferred option. (Message 16 as well)
  3. Some of the other relatively more preferred messages are Message 6 (Efficacy comparison (vs efavirezen)) and Message 5 (Efficacy comparison (vs darunavir))
  4. Physicians are comparatively more neutral to Message 2 (Resistance rate comparison) and Message 8 (Managing patients with HIV and hyperlipidemia)
  5. Relatively high degree of variance is observed in Message 1 (Efficacy in presence of NNRTI resistance mutations), Message 8 (Managing patients with HIV and hyperlipidemia).
  6. While fewer differences are observed based on the type of physicians relative differences can be observed for Canadian Physician compared to Mexican Physicians.
  • There is some issue with the newest version of mlogit especially while estimating a latent class model using gmnl function of gmnl package so here we download a older version of the package instead.
